Introduction

The purpose of this project is to create an application that helps bicyclists find the best routes with the least elevation in San Francisco.

Project development

Since the inception of my maze game project in August, the journey has been both challenging and rewarding. In the initial weeks, I focused on laying a solid foundation by creating a Makefile and a header file, which were essential for organizing and compiling the project efficiently. This phase involved a lot of planning and collaboration with my team to ensure that we were all on the same page regarding the project’s structure and goals. We held regular meetings to discuss our progress and address any roadblocks, which helped us stay aligned and motivated. By early September, we had made significant strides. I successfully downloaded the necessary game assets, which added a visual and auditory dimension to our project. Additionally, I created a detailed map of the maze, which served as a blueprint for the game’s design and functionality. This period was marked by intense collaboration, as we shared ideas and feedback to refine the maze’s layout and features. The team’s collective effort and dedication have been instrumental in bringing the project to life, and we continue to work closely to ensure its success.

Why I Chose This Project

The inspiration for my maze game project stems from a combination of my passion for puzzles and my fascination with game design. Growing up, I was always captivated by the intricate designs and the sense of accomplishment that came with solving mazes. This project is an opportunity to bring that same excitement to others. I wanted to create a game that not only challenges players but also provides a visually engaging and immersive experience. The idea of navigating through complex pathways, encountering obstacles, and discovering hidden secrets within the maze excites me. Additionally, I aim to incorporate various levels of difficulty to cater to both casual players and hardcore puzzle enthusiasts.

Project Accomplishments

Our application helps urban cyclists find optimal routes in real-time.

Technologies Used

  • SDL2: For rendering and handling user input.
  • C Language: For its performance and low-level capabilities.
  • GCC: As the compiler, ensuring compatibility with Ubuntu 14.04 LTS.
  • JSON: For data serialization.
  • HTTP API: For managing user data and level.

Key Features

  • Real-time weather updates for cyclists.
  • Responsive design compatible with mobile devices.

Most Difficult Technical Challenge

Early in the project, I decided to implement Facebook OAuth to allow users to save their routes. The project required SSL security for OAuth, which we had to deploy under a tight deadline. By collaborating with peers and seeking external support, we managed to secure SSL and implement the feature in time.

What I Learned

As I delved into the development of my maze game project, I quickly realized the critical role that algorithms and data structures play in creating an engaging and efficient game. The challenge of dynamically generating mazes led me to explore various maze generation algorithms, such as Depth-First Search (DFS) and Prim’s algorithm. Implementing these algorithms not only enhanced my understanding of how they work but also allowed me to appreciate the elegance and efficiency they bring to problem-solving. The process of coding these algorithms and seeing them come to life in my game was incredibly rewarding and sparked a deeper interest in the field of algorithms. Additionally, the need for efficient pathfinding within the maze introduced me to pathfinding algorithms like A* (A-star) and Dijkstra’s algorithm. These algorithms are essential for determining the shortest path through the maze, ensuring that the game remains challenging yet solvable. Working with these algorithms required me to think critically about data structures such as priority queues and graphs, which are fundamental to their implementation. This hands-on experience with algorithms and data structures not only improved my coding skills but also deepened my appreciation for their importance in computer science and game development.
